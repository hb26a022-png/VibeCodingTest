# ゲロケロフロッピー 仕様書

本仕様書は、他システムへの連携およびAIへの提示を目的とした、サイドスクロール式カエルアクションゲームの構成・動作仕様書です。

---

## 1. ゲーム概要
プレイヤーは主人公のカエル「フロッピー」を操作し、様々な性質を持つヘビたちの包囲網や過酷な沼地のトラップを乗り越え、ゴールを目指すレトロスタイルの2D横スクロールアクションゲームです。本ゲームは「ノーコンティニュー連続クリア」による隠し要素など、何度も失敗と挑戦を繰り返してプレイスキルを磨いていく、やや高めの難易度（死にゲー・覚えゲー）に設定されています。

---

## 2. プレイヤー「フロッピー（カエル）」の仕様・アクション
プレイヤーは、以下の基本パラメータおよびアクションを持ちます。

### 2.1 プレイヤーの状態（パワーアップ）
アイテムを入手することで、フロッピーの姿と能力が3段階に変化します。
- **NORMAL（通常形態）**:
  - 初期の姿。攻撃は通常の長さの舌攻撃と踏みつけ。ダメージを受けると1デスとなります。
- **SUPER（巨大形態）**:
  - 「ゴールデン・バグ」を入手すると変化する青くて大きな姿。
  - 舌の攻撃レンジが大幅に拡張され、レンガブロックを頭突きで破壊できます。
  - ダメージを受けた際、NORMALに退化するだけで即死を防げます。
- **FIRE（バブル形態）**:
  - 「バブル・パワー」を入手すると変化する赤い姿。
  - 地上をバウンドする「水泡（バブル）弾」を発射して、敵を遠距離から倒せます。

### 2.2 基本アクション
1. **左右移動**: レトロゲーム同様の慣性がある移動です。
2. **ジャンプ**: ボタンを押し続ける長さによって高さが調節できる跳躍です。
3. **舌伸ばし攻撃**:
   - 舌を前方に伸ばし、接触した敵を巻き込んで反転（Flip）させて倒します。
   - 舌でブロックを叩いてアイテムを出すことも可能です。
4. **踏みつけ**: 敵の頭上に落ちることで踏みつけ（Squish）ダメージを与えられます。
5. **水上ジャンプ**: 水面に接地した瞬間にジャンプすることで、水上を撥ねて跳ぶことができます。

## 3. ステージ構成・オブジェクト
通常面とボスステージを合わせて計4つのステージが存在します。

### 3.1 通常ステージ (Stage 1 〜 3)
- 各ステージの最奥部には、ゴール標識（**巨大なスイレンの花が飾られたポール**）が配置されています。
- プレイヤーがポールに触れる（到達する）ことでステージクリアとなります。

### 3.2 隠しステージ：ボスエリア (Stage 4)
- **解放条件**:
  - **Stage 1からStage 3までの3ステージを、1度もゲームオーバーを挟まずに連続でクリアすること**。
  - ※途中でゲームオーバーになりコンティニューした場合は解放されず、そのままStage 3クリア時に通常エンディングとなります。
- **仕様**:
  - ボスエリアには**ゴール用のポールが存在しません**。
  - エリア内に鎮座するボス「ヘビの大王（キングスワンプスネーク）」を完全に撃破した瞬間に、ゲームクリア（真の完全制覇）となります。

### 3.3 配置オブジェクト＆トラップ
- **GROUND（地面ブロック）**: 通常の足場。
- **BRICK（レンガブロック）**: SUPER状態で下から叩くと破壊可能。
- **QUESTION（ハテナブロック）**: 叩くとアイテム（ハエ、ゴールデン・バグ、バブル・パワー、1UPキノコなど）が出現。
- **USED（使用済みブロック）**: トークン排出後のハテナブロック。
- **SOLID（堅いブロック）**: 破壊不可能な地形ブロック。
- **TRUNK & TRUNK_TOP（木の幹）**: いわゆる土管に相当する遮蔽地形。
- **WATER（泥水・沼）**: カエルが最大2.0秒間だけ上に乗って歩くことができる特殊ハザード。水に触れている間は頭上にスタミナゲージ（残り秒数カウントダウン）が表示され、累計2.0秒（120フレーム）に達すると沈んで溺死（1ミス）します。乾いた安全な足場（GROUND等）に着地した瞬間に歩行制限時間は完全にリセットされますが、水面をジャンプして飛び跳ねているだけでは制限時間は回復しません。
- **SPIKES（トゲ）**: 触れるとダメージを喰らう障害物。SUPERやFIRE状態であれば段階的なパワーダウンと一定時間の無敵が付与され、NORMAL状態であればミス（1デス）となります。接触した際はノックバック（跳ね返り）が発生し、一定時間すり抜けられます。

## 4. 敵キャラクター（ヘビ）一覧

| 敵の名前 | 種類 (EnemyType) | 特徴・行動パターン |
| :--- | :--- | :--- |
| **みどりヘビ** | `SNAKE_GREEN` | 地上を終始徘徊する最も基本的なヘビ。崖があるとそのまま落下します。 |
| **あかヘビ** | `SNAKE_RED` | 移動速度が速く、足場の崖を検知して手前で自動反転するため落下しません。 |
| **はねヘビ** | `SNAKE_WINGED` | 常に細かく跳ね回り、プレイヤーに予期せぬ位置から接触を狙う飛行型のヘビ。 |
| **ヘビの大王** | `SNAKE_BOSS` | 隠しステージに君臨する巨大な紫色のヘビ（冠を戴いている）。<br>**耐久力（HP）は 5**。踏みつけることでダメージを与えます。被ダメージ時に移動速度が跳ね上がります。 |

## 5. ゲームバランスと高難易度設計
本作品は、以下の要素により「何度もやり直すスリルと達成感」を提供するバランス設計になっています。

- **シビアな敵配置とトラップ**:
  トゲ（SPIKES）や水域（WATER）の隙間をダッシュジャンプで縫うように進む場所が多く、初見での突破は非常に難しく設計されています。
- **チェックポイントなし仕様**:
  ミスをした場合やゲームオーバー時は、ステージの最初からやり直し。3ステージ連続ノーデスクリアという極限状態でのみ辿り着けるボス戦は、プレイヤーに最大級の緊張感を提供します。
- **ボス「ヘビの大王」の高い戦闘力**:
  巨大な体躯と、ダメージを受けるほど機敏になる反撃行動により、完璧な踏みつけのタイミングと、エリア内のパワーアップブロック（ゴールデンバグ、バブルパワー）の戦略的な回収が必要不可欠となります。
